# Design 1 : "Dopamine & Neon"

> Design System sombre avec accents néon pour SHEED

## Palette de Couleurs

| Rôle | Couleur | Hex Code |
|------|---------|----------|
| **Fond Principal** | Noir Profond | `#101012` |
| **Fond Surface** | Gris Foncé | `#1C1C1E` |
| **Primaire (Gradient)** | Pink → Purple | `#FF3B7A` → `#9A3BFF` |
| **Accent / Succès** | Vert Néon | `#00F5A0` |
| **En Attente** | Jaune Ambre | `#FFC700` |
| **Texte Principal** | Blanc Cassé | `#F5F5F7` |
| **Texte Secondaire** | Gris Clair | `#8A8A8E` |
| **Bordure** | Gris Discret | `#3A3A3C` |

## Typographie

- **Titres** : Satoshi (Bold, Black)
- **Corps & UI** : Inter (Regular, Medium, SemiBold)

## Style Visuel

- Mode sombre avec accents néon
- Glassmorphism (transparence + blur) sur les cards
- Coins très arrondis (`rounded-3xl`)
- Boutons en forme de pilule avec gradient
- Effet glow subtil sur les éléments interactifs
- Inspirations : Tinder, Bumble, Hinge, BeReal, TikTok

## Vibe

Premium, moderne, Gen-Z, addictif, social
